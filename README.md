README - Projeto do Agente de IA - Passo a Passo

-Instalar dependências:
Na pasta principal Aula_Agente_IA-main execute:
npm install

-Iniciar o backend
Na pasta principal execute:
node server.js

-Iniciar o frontend
Abra outro terminal, entre na pasta agente e execute:
cd agente >
npm install >
npm run dev

-Executar os testes
Na pasta principal execute:
npx jest validarMensagem.test.js
